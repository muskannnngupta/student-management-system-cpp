# 🎓 Student Management System (C++)

A simple Student Management System built using C++ and File Handling.

## 🚀 Features

* ➕ Add Student (Name + Marks)
* 📖 View All Students
* 🔍 Search Student by Name
* ❌ Exit Program

## 📂 File Storage

All student data is stored in a file:

```
students.txt
```

Example:

```
Muskan : 90
Riya : 85
Aman : 78
```

## 🛠️ Technologies Used

* C++
* File Handling (`fstream`)
* Standard Input/Output

## ▶️ How to Run

1. Compile the program:

```
g++ main.cpp -o main
```

2. Run the program:

```
./main
```

## 💡 Concepts Used

* File Handling (Read/Write)
* Loops and Conditions
* String Handling
* Menu-driven Programming
